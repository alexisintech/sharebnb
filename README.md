## Built With

- React.js, Webpack, Babel
- HTML5/CSS3, Javascript ES6
- ESlint, Stylelint
- VSCode

## Get your own copy running!

- Clone using your preferred method 
- `npm i` 
- `npm start`

The page will reload if you make edits.<br />
You will also see any lint errors in the console.
